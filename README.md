## :rocket: Installation

```
git clone https://github.com/ishbguy/brof
cd brof
export PATH="$PATH:$(pwd)/bin"
```

or

```
curl -fLo /path/to/brof.sh \
        https://raw.githubusercontent.com/ishbguy/brof/master/bin/brof.sh
export PATH="$PATH:$ /path/to"
```
